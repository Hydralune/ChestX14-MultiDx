# 医学图像多标签分类课程作业要求

## 一、作业目标

胸部 X 光检查是诊断呼吸系统和心脏疾病的首选工具。随着深度学习的快速发展，
胸片疾病智能识别成为提升医疗效率的重要技术方向。

本课程作业要求学生使用授课教师提供的 **胸部 X 光多标签图像数据集**，
共 **21844 张正位 X 射线图像**，**14 类标签**，构建深度学习模型，实现疾病的智能辅助诊断。

------

## 二、作业内容要求

### 1. 模型开发与提交（代码 + 模型文件）

- 包含 **数据预处理、模型构建、训练、验证与测试** 的完整流程
- 提交内容需包括：
  - 源代码
  - 环境配置文件
  - 训练完成的模型权重文件
- 模型需使用以下两个指标进行性能评价：
  - **AUC**
  - **F1-score**

------

### 2. 课程报告（PDF 或 Word）

报告需包含以下内容：

- 问题背景简述
- 数据分析与预处理方式
- 模型设计与训练策略说明
- 结果分析与可视化展示
  - 如 ROC 曲线
  - 混淆结果统计等
- 模型不足与改进方向

------

### 3. 展示（答辩）

- 每组进行 **5–10 分钟** 展示
- 展示内容包括：
  - 解决方案
  - 模型亮点
  - 性能结果
- 回答老师与同学提问

------
